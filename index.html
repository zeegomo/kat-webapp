Help Guide

Step 1: Test Information

Enter details about your test:

  • Event name: A unique identifier for this test session
  • Substance: What you expect the sample to be
  • Appearance: Physical form (powder, crystal, pill, etc.)
  • Photo: Required for pills, capsules, and blotters

Step 2: Calibration

Verify the spectrometer is ready:

  • Open the filter bay before starting
  • Tap "Start" to see the live camera preview
  • Verify the laser line is visible and centered
  • Close the filter bay before proceeding

Step 3: Capture

Acquire and review spectra:

  • Adjust shutter time (0.1-10s) and gain as needed
  • Tap "Capture" to acquire a spectrum
  • Review the identification result and confidence score
  • Capture multiple spectra if needed
  • Export results when finished

Connection Issues

Not connected to spectrometer:

  • Go to your device's WiFi settings
  • Connect to the network named "spettromiao"
  • Wait a few seconds for the connection
  • Reload the app if the warning persists
  • Verify the spectrometer is powered on

Connected to WiFi but still shows "Not connected":

  • Open https://192.168.4.1 directly in your browser
  • Accept the self-signed certificate warning
  • Return to the app and reload

Capture Problems

Camera timeout / stuck:

  • Reload the page, or go back and forward a step
  • Restart the spectrometer device
  • Try reducing shutter time (e.g., 3s instead of 5s)
  • Wait for the camera to fully initialize after restart

Capture failed:

  • Check that the spectrometer is still connected

Capture too bright / dark:

  • Check that the filter bay is closed
  • Try a lower gain setting if image is overexposed
  • Increase shutter time or gain if image is too dark

Save/Export Issues

Failed to save or export:

  • Check if browser storage is full
  • Export and delete old tests from History
  • Try using a different browser
  • Clear browser cache if issues persist

Sync Issues

Sync not working:

  • Verify the server URL is correct in Settings
  • Check that your API token is valid
  • Ensure you have internet connectivity
  • Use "Test Connection" to diagnose issues

Storage Access Issues

"Storage access denied" on startup:

  • Make sure you're not in incognito/private browsing mode
  • Open Chrome Settings → Site Settings → Cookies and site data
  • Ensure "Allow sites to save and read cookie data" is enabled
  • Check that this site isn't blocked under "Sites that can never use cookies"
  • Reload the app after changing settings
